### 🌸 О проекте

**Habit Tracker** - это стильное веб-приложение для отслеживания ежедневных привычек, выполненное в нежно-розовой гламурной тематике.  

Проект создан в рамках практической работы по организации мини-проекта с использованием **IDE, Git, GitHub и GitHub Projects**.

### Особенности

- Нежнейшая розовая градиентная тема с сердечками и плавными анимациями
- Добавление, удаление и отслеживание привычек
- Автоматический расчёт **стрика** (дней подряд)
- Красивая статистика в реальном времени
- Полностью на **Python + Flask**
- Сохранение данных в `habits.json`
- Адаптивный дизайн (отлично смотрится на любом устройстве)

### Как запустить

#### 1. Клонируй репозиторий
```bash
git clone https://github.com/alessstft/habit_tracker.git
cd habit_tracker
```
2. Установи зависимости
bash
``` pip install -r requirements.txt```
4. Запусти приложение
bash
```python app.py```
5. Открой в браузере
Перейди по ссылке:``` http://127.0.0.1:5000```


### Технологии

Backend: Python + Flask
Frontend: HTML5, CSS3, JavaScript
Хранение данных: JSON
Стиль: Нежно-розовая гламурная эстетика
